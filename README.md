# 📊 Zomato Customer Analytics using BigQuery SQL

## 📌 Overview

This project demonstrates an end-to-end Data Analytics workflow using **Google BigQuery**. The objective is to analyze Zomato customer, restaurant, and order data to uncover business insights through data cleaning, exploratory data analysis (EDA), and SQL-based analytics.

The analysis focuses on identifying revenue trends, customer behavior, restaurant performance, coupon usage, cancellations, refunds, and customer churn to support data-driven business decisions. :contentReference[oaicite:0]{index=0}

---

## 📂 Dataset

The project uses three datasets:

- **Orders** – Order details, payments, coupons, refunds, delivery status
- **Customers** – Customer information and acquisition details
- **Restaurants** – Restaurant details, cuisine, ratings, and city

## 📈 Project Workflow

### 1. Data Loading
- Imported CSV datasets into Google BigQuery
- Verified schema and data types
- Created relational tables

### 2. Exploratory Data Analysis (EDA)
- Checked dataset structure
- Identified missing values
- Detected duplicate records
- Reviewed null values and inconsistencies
- Explored distributions and key metrics

### 3. Data Cleaning
- Removed duplicate records
- Handled missing values
- Corrected inconsistent data
- Standardized column formats
- Validated relationships between tables

### 4. SQL Analysis

Performed SQL queries to answer business questions such as:

- Total Revenue
- Monthly Revenue Trend
- Top Revenue-Generating Cities
- Average Order Value (AOV)
- Top Customers by Revenue
- Repeat Customer Analysis
- Restaurant Performance
- Most Popular Cuisines
- Coupon Usage Analysis
- Cancellation & Refund Analysis
- Customer Churn Analysis :contentReference[oaicite:1]{index=1}

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/RohitChxudhary/Zomato_Data_Analysis
```

2. Open **Google BigQuery**

3. Create a new dataset

4. Import the CSV files into BigQuery tables

5. Run the SQL scripts in the following order:
   - Data Cleaning
   - EDA
   - Business Queries

6. Review the query outputs and insights.
